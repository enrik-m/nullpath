## 2. The service

Nullpath is a single-page web application that helps you track your
offensive-security learning progress. You complete nodes, attach
resources, write notes, log bounty submissions, and earn achievements.

exploits**, **does not interact with any third-party system on your
behalf**, and **does not host hacking labs**. It's a personal tracker
with a curated taxonomy. The actual learning happens elsewhere
(PortSwigger Web Security Academy, HackTheBox, TryHackMe, books, CTFs).
